# OIBSIP_python_4
Basic weather app

# Objective
Create a weather app in python that fetches and displays real-time weather data for a user-specified location using weather API.

# Project level
--**Beginner**: Command-line interface

# Steps performed
- User inputs city name or ZIP code
- Fetches current weather data(temperature, humidity, description, wind speed)
- Displays weather conditions clearly
- Error handling for invalid locations or API issues

# Tools and libraries
- Pyhton 3.x
- 'requests' - for API calls
- 'json' - for parsing data

# Outcome
Successfully built a functional and user-friendly weather application that provides accurate real-time and forcast data, helping users plan their day effectively.
